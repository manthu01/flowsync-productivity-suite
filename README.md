# FlowSync Productivity Suite

A full-stack task management dashboard built using:

- React
- Tailwind CSS
- Node.js
- Express.js
- MySQL

## Features

- Create tasks
- Edit tasks
- Delete tasks
- Task filtering
- Search functionality
- Dashboard analytics
- Responsive UI

## Tech Stack

Frontend:
- React
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js
- MySQL
